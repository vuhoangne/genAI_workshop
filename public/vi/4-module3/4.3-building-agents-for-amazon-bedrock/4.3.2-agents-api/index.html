API của Agent

Bạn cũng có thể gọi agent của mình thông qua API.

Để thử nghiệm:

  1. Quay lại VSCode Server và mở tệp rag_examples/agent_rag.py.
  2. Cập nhật AGENT_ID với ID của Agent của bạn. ID này nằm trong phần tổng quan của Agent mà bạn đã tạo.

  1. Nhấp vào biểu tượng Menu (hình hamburger) ở góc trên bên trái.
  2. Điều hướng đến Terminal -> New Terminal.
  3. Chạy mã với lệnh python3 rag_examples/agent_rag.py.
  4. ️Thử thay đổi QUERY ở dòng 6 để xem các loại phản hồi khác nhau.
